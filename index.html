HTML

  • The head element contains metainformation about the webpage.
  • Semantic html is html that describes what it's doing
  • The body element represents the visible content shown to the user.

CSS

  • Inline CSS styles apply to a specific HTMP tag
  • Internal style sheets contain CSS rules in the head section of the HTML file
  • External CSS style sheets allow developers to keep all rules in a separate sheet
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A class is a group of elements that are the same or similar.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • How to create a new branch:
    1. In the command line, navigate to the local directory.
    2. Check what branch you are currently on by running git status.
    3. If you are not on the main branch, run git checkout main to switch to the main branch.
    4. Pull the remote main branch into the local main branch by running git pull origin main.Now your local and remote main branches are in sync!
    5. Create a new feature branch by entering git checkout -b feature/... into the command line.
  • How to upload and commit a new branch:
    1. Add changed files in our working directory to the staging aread of git by entering git add -A
    2. Commit to our repository with git commit -m "add description of changes"
    3. Ensure the current branch is up to date before opening a pull request with git pull origin main
    4. Push up all changes by entering git push origin feature/...
    5. Navigate to the remote repository in GitHub. Click "Pull Requests" > "New pull request"
    6. Make sure the base branch is main and the comparison branch is the feature/... branch. Add a title for the pull request along with a short description of the changes
    7. Click the "Create pull request" button
    8. Now merge the comparison branch into the base branch by selecting "Merge pull request" button and then confirming
    9. Lastly, update our local main branch with entering git checkout main in the command line
    10. Puill the remote main branch into the local main branch by entering git pull origin main

JavaScript

  • Note that whereas the style.css file is linked within the head element, the script.js file is linked within the body element.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A standard for loop starts with for(var x = 0; x < array.length; x++) {code block}
